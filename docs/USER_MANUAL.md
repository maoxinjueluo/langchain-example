# 用户操作手册（MVP）

## 1. 注册与登录

1. 打开 `/register`，填写邮箱、姓名、密码。
2. 收到验证邮件后点击链接完成邮箱验证。
3. 在 `/login` 输入邮箱密码登录，可勾选“记住我”。
4. 忘记密码使用 `/forgot` 发起重置。

## 2. 管理知识库（管理员）

1. 登录后进入“知识库管理”。
2. 新建知识库：填写名称、可见范围、分类、标签。
3. 在知识库详情页上传文档（TXT/PDF/DOCX，<=50MB）。
4. 系统自动执行：MD5 去重 -> 解析 -> 分块 -> 向量化。
5. 可在详情页查看处理状态、错误信息，并可删除文档。

## 3. 智能问答

1. 进入“智能问答”页面。
2. 选择知识库并输入问题。
3. 系统基于知识库检索并生成回答。
4. 可查看引用来源、置信度。
5. 对回答可执行收藏与反馈评价。

## 4. 常见问题

- 上传失败：检查格式与大小限制。
- 低置信度：说明检索证据不足，建议补充资料或改写问题。
- 未收到邮件：未配置 SMTP 时系统会在服务日志中输出邮件内容。
